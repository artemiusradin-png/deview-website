# DeView — AI Enterprise Infrastructure

A modern marketing website for DeView, an AI enterprise infrastructure services startup.

## Design

- **Colors**: Black (#0a0c0b), grey (#aab5ae), green (#43c97b) — matching Task Manager App reference
- **Font**: IBM Plex Sans
- **Animations**: Framer Motion for scroll-triggered and hover animations

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
